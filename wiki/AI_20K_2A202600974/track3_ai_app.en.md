# Track 3: AI Application (Agent Developer)


*Track 3 focuses on the **Application and Development Aspects** of an AI system. Learners in this Track play the role of an AI Developer / Agent Developer - the person who directly writes code, builds logic for AI Agents, integrates tools, and packages them into complete applications.*


*- **Agent Frameworks**: Building Agents using LangChain, LangGraph, AutoGen (See also: [[concept_agent]]).*
*- **Tool Calling & MCP**: Writing functions for the Agent to use, connecting the Agent to tools via the Model Context Protocol.*
*- **RAG Implementation**: Implementing an Advanced RAG pipeline with code (See also: [[concept_rag]]).*
*- **Guardrails**: Implementing safety mechanisms, prompt/response moderation using NeMo Guardrails.*
*- **Agentic Routing**: Routing logic between multiple Agents in a Multi-Agent system.*
