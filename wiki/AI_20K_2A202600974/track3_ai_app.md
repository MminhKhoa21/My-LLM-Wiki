# Track 3: AI Application (Agent Developer)

Track 3 tập trung vào **Khía cạnh Ứng dụng và Lập trình (Application & Development)** của một hệ thống AI. Người học ở Track này đóng vai trò là AI Developer / Agent Developer - người trực tiếp viết code, xây dựng logic cho AI Agent, tích hợp các công cụ (Tools), và đóng gói thành ứng dụng hoàn chỉnh.

## Các chủ đề cốt lõi
- **Agent Frameworks**: Xây dựng Agent bằng LangChain, LangGraph, AutoGen (Xem thêm: [[concept_agent]]).
- **Tool Calling & MCP**: Viết các function cho Agent sử dụng, kết nối Agent với các công cụ qua Model Context Protocol.
- **RAG Implementation**: Triển khai Advanced RAG pipeline bằng code (Xem thêm: [[concept_rag]]).
- **Guardrails**: Implement cơ chế an toàn, kiểm duyệt prompt/response bằng NeMo Guardrails.
- **Agentic Routing**: Logic định tuyến giữa nhiều Agent trong hệ thống Multi-Agent.
